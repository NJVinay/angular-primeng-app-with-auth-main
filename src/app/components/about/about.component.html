About This Project

This is a modern to-do list application designed to showcase a variety of web development skills. It is built with Angular and PrimeNG and features a clean, intuitive UI with a glassmorphism theme. The goal of this project is to demonstrate proficiency in front-end development, including component-based architecture, reactive forms, and API integration.

Key Features:

  • Create, read, and delete to-do items
  • Mark items as completed with a checkbox
  • User authentication (login and registration)
  • Modern and responsive design with a glassmorphism theme
  • Built with Angular and PrimeNG for a rich UI
